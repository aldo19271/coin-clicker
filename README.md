# 🪙 Gold Coin Clicker

Gold Coin Clicker adalah game clicker sederhana bertema koin emas.  
Pemain harus mengetuk koin untuk mengumpulkan gold dan membuka berbagai skin unik.

## ✨ Fitur Game

- 🪙 Klik koin untuk mendapatkan gold
- ⚡ Random events spesial
- 🔥 Combo multiplier system
- 🌮 TACOS RAIN mode
- 🛒 Shop skin coin
- 🎵 Sound effects interaktif
- 🌙 Midnight bonus
- 💾 Auto save progress
- 📱 Tampilan mobile friendly

## ⚡ Random Events

- 💥 Double Coins → coin x2 sementara
- 🎰 Jackpot → bonus x10
- 🔥 Frenzy Mode → klik x3
- ✨ Golden Rain → coin otomatis
- 🌮 Tacos Rain → +50 per klik dengan musik taco
- 🌙 Midnight Bonus → bonus malam hari

## 🛒 Skin Coin

Game memiliki beberapa skin:
- Gold Classic
- Silver Moon
- Ruby Flame
- Emerald Forest
- Diamond Ice
- Cosmic Void

## 🎮 Cara Bermain

1. Klik koin untuk mendapatkan gold
2. Kumpulkan sebanyak mungkin coin
3. Beli skin baru di shop
4. Manfaatkan random events untuk mendapatkan bonus besar
